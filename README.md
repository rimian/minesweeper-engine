# Minesweeper

[![Build Status](https://travis-ci.org/rimian/minesweeper.svg?branch=master)](https://travis-ci.org/rimian/minesweeper)

A Minesweeper engine.

## Run in console

Load into console
```
node
> .load cli
```

Start the game
```
start()
```

Expose a tile
```
expose(x, y)
```

## Testing

`yarn test`
